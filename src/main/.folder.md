# src/main (主进程模块)
划分 Electron 后台能力的独立模块，避免 `main.js` 变成大杂烩。

- `ipc-handlers.js`: 集中注册和处理 Renderer 层的 IPC 通信请求。
- `logger-service.js`: 后台日志轮转和记录服务。
- `system-service.js`: 本地系统交互服务（如字体扫描等）。
- `window-manager.js`: 窗口的创建、参数配置和生命周期管理。
