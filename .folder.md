# Root Directory
The core entry points and configuration for the RZFrame Electron application.

- **main.js**:  [Entry] Electron Main Process. Handles native I/O, ExifTool management, and IPC.
- **preload.js**: [Bridge] Secured IPC bridge with whitelisted channels.
- **renderer.js**: [Entry] Main Renderer orchestration (Canvas rendering, State management).
- **index.html**: [Entry] UI Template (Tailwind + Native DOM).
- **src/**:  [Source] Hierarchical source code (core drawing logic, UI modules).
- **assets/**: [Resource] Static assets including builtin Brand Logos (`assets/logos/Svg`).
- **TODO.md**: [Docs] Roadmap and future enhancement ideas.
- **CHANGELOG.md**: [Docs] Version history (currently v2.7.3).
- **README.md**: [Docs] Main entry point for developers and photographers.
- **docs/**:   [Docs] Analysis, PRDs, and quality audit reports.
- **RELEASES/**: [Release] Final installer artifacts.
- **logs/**:   [Data] Runtime error logs.
