# Root Directory
The core entry points and configuration for the RZFrame Electron application.

- **main.js**:  [Entry] Electron Main Process. Handles lifecycle, window creation, and native IPC.
- **preload.js**: [Bridge] Safe bridge between Main and Renderer processes (Context Isolation).
- **renderer.js**: [Entry] Main Renderer logic (Orchestration Layer).
- **index.html**: [Entry] Application UI entry point.
- **src/**:  [Source] Core application logic, UI components, and utilities.
- **assets/**: [Resource] Static assets (icons, logos).
- **dist/**:   [Build] Bundled output (Vite).
- **logs/**:   [Data] Application runtime logs.
- **docs/**:   [Docs] Project documentation center.
