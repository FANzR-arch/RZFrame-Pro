# RZFrame Workstation Pro - 产品介绍书

> **赋予每一张照片应有的仪式感**

## 1. 产品概述 (Overview)
**RZFrame Workstation Pro** 是一款专为摄影师和摄影发烧友打造的桌面端 EXIF 边框生成工具。不同于简单的拼图软件，RZFrame 深度集成了专业的元数据解析引擎，能够自动读取并展示照片背后的拍摄参数（光圈、快门、ISO、镜头型号等），将枯燥的数据转化为影像叙事的一部分。

我们相信，参数不仅是记录，更是摄影师思考的注脚。RZFrame 用极简的设计语言，帮助你将作品以最专业的姿态呈现给世界。

## 2. 核心价值 (Core Values)

### 💎 极致画质 (Lossless Quality)
拒绝压缩。RZFrame 采用全流程无损处理管线，支持加载高达 50MP+ 的高分辨率原图，通过 Canvas 级像素渲染，确保导出的每一张作品都不仅保留了原图的细腻质感，更拥有了画廊级的展示效果。

### 🚀 专业数据解析 (Pro EXIF Engine)
内置工业级 `exiftool-vendored` 引擎，支持 Sony, Canon, Nikon, Fujifilm, Leica, Hasselblad 在内的数千种相机与镜头组合。
- **智能镜头识别**: 自动解析复杂的镜头 ID，告别 "Unknown Lens"。
- **全参数展示**: 焦段、光圈、快门速度、感光度、拍摄时间精确呈现。

### 🔒 本地隐私安全 (Privacy First)
您的照片属于您自己。RZFrame 是纯本地运行的桌面应用 (Electron)，无需上传云端，无需担心照片泄露。所有处理均在您的计算机上完成，安全、私密、快速。

## 3. 核心功能 (Key Features)

### 3.1 沉浸式模版系统
提供三款精心打磨的基础模版，满足不同风格的表达需求：

- **🖼️ Classic (经典画廊)**
    - 最纯粹的展示方式。底部留白，清晰罗列拍摄参数，如同悬挂在画廊墙上的艺术微喷。
    - *适用场景*: 风光、人像、纪实摄影。

- **🎬 Cinema (电影叙事)**
    - 上下黑边设计，模拟 2.35:1 宽银幕电影质感。参数以字幕形式隐约浮现，营造强烈的叙事氛围。
    - *适用场景*: 街头摄影、黑白胶片、具有故事感的画面。

- **☁️ Float (悬浮投影)**
    - 独特的悬浮阴影设计，配合高斯模糊背景，让照片仿佛漂浮在屏幕之上，极具立体感与现代感。
    - *适用场景*: 现代建筑、极简构图、社交媒体分享。

### 3.2 高自由度的定制化
拒绝千篇一律，每一个细节都可由您掌控：
- **画幅调整**: 内置 1:1, 4:5, 16:9, 9:16 等社交媒体黄金比例，支持原图缩放与平移。
- **字体管理**: 支持调用系统本地字体，从衬线体的优雅到无衬线体的干练，随心切换。
- **品牌展现**: 支持上传个人 Logo 或输入品牌文字，打造个人专属 IP。
- **参数微调**: 觉得自动读取的镜头名字太长？点击即可直接编辑修改。

### 3.3 高效生产力工具
- **批量处理**: 无论是一次旅行的精选集，还是一组胶片扫描档，直接拖入软件，一键批量生成。
- **配置同步**: 一键将当前精心调好的参数（边框宽度、字体大小、配色）应用到列表中的所有图片。
- **主题切换**: 适配系统深色/浅色模式，深夜修图也不刺眼。

## 4. 适用人群
- **职业摄影师**: 需要快速输出带有参数的样片，用于作品集展示或客户交付。
- **摄影博主/KOL**: 在小红书、Instagram 等平台分享高质量作品，建立专业形象。
- **胶片爱好者**: 记录每一卷胶片的拍摄参数与底片扫描件，数字化归档。
- **器材评测者**: 清晰展示不同镜头与机身的样片参数。

## 5. 结语
RZFrame Workstation Pro 不仅仅是一个工具，它是您摄影工作流的最后一块拼图。它让分享变得简单，让专业变得可见。
