# Docs (文档中心)
存放项目所有的核心文档，包括需求、计划、审计报告等。

- `AUDIT_DEEP_[DATE].md`: 全维度产品/技术审计报告。
- `GROWTH_PLAN_[DATE].md`: 增长与商业化战略计划。
- `PRD.md`: 产品需求文档 (Product Requirement Document)。
- `Product_Introduction.md`: 产品介绍文案。
