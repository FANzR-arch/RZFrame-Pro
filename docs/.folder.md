# Docs (文档中心)
存放项目所有的核心文档，包括需求、计划、审计报告等。

- `AUDIT_DEEP_[DATE].md`: 全维度产品/技术审计报告。
- `GROWTH_PLAN_20260202.md`: 商业化与增长战略 (Response to Soul Torture)。
- `Audit_Report.md`: 早期/简要审计报告。
- `CODE_REVIEW.md`: 技术代码审查详情。
- `PRD.md`: 产品需求文档 (Product Requirement Document)。
- `Product_Introduction.md`: 产品介绍文案。
- `SHOWCASE_RZFRAME.md`: 产品复盘与展示文档 (5-Section Story)。
