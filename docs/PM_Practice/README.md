## 💡 常见问题 (FAQ)

**Q: 我一定要把所有表格都填满吗？**
A: 不需要。如果是空的内容，就写“暂无数据”。**真实**比**完整**更重要。

**Q: 找不到用户访谈怎么办？**
A: **Be Creative.** 去小红书私信、去摄影群潜水、甚至找你的朋友扮演用户。作为 PM，你的第一项能力就是“搞定资源”。

**Q: AI 写代码卡住了怎么办？**
A: 回到 PRD。通常是因为你没有把需求拆解得足够细。试着把 PRD 里的一个大功能拆成 3 个小 Steps，再发给 AI。
