## 🎯 使用场景

**场景 1：准备用户访谈**  
→ 阅读 `01_用户访谈问卷模板.md`  
→ 使用 `02_用户访谈记录表.md` 进行访谈  
→ 完成后填写 `03_用户访谈分析报告模板.md`

**场景 2：竞品分析**  
→ 按照 `04_竞品分析执行指南.md` 收集信息  
→ 填写 `05_竞品对比矩阵模板.md`

**场景 3：撰写 PRD**  
→ 参考 `07_PRD_v2.0_模板.md`  
→ 使用 `08_功能优先级评分表.md` 排定优先级
